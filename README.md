# Hangman Game
This Program generates random words through the wonderwords library, then conceals the letters using underscores. 
It asks the user to guess the contents of the word,  letter by letter.  The game ends when either, the user guesses 
the word or runs out of chances.  

## How to Run
### In a terminal

1. Go to the project root directory, and install the required wonderwords library.
```
   pip install wonderwords
```

2. Run the program.
```
   python main.py
```

### In Google Colab

1. Run this in a code cell to install the required wonderwords library.
```
   !pip install wonderwords
```

2. Paste the code in main.py into a code cell.  After the previous code cell finishes, run this code cell.

---
When this program is run for the first time without previously installing Wonderwords.  You will get an error 
code that will link you to the Wonderwords download.
